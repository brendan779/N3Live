== NOTES ==
- RNDIS network function (IF0+IF1) → macOS binds it as en3/en4 but reports
  "media inactive" and never completes DHCP. The macOS RNDIS stack does not
  bring the link up on its own.
- IF2 mass storage → the goggles' storage mounts as a disk.
- IF3–IF7: five DJI vendor-specific interfaces (0xFF/0x43/0x01), each a
  bulk OUT + bulk IN pair, 512-byte packets. No macOS kernel driver is bound
  to these, so they are claimable directly via libusb. One of these almost
  certainly carries the live H.264/H.265 video; the others are likely
  control (DUML), telemetry/OSD, audio and a mobile/MFI channel.
